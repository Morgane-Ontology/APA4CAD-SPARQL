# APA4CAD-SPARQL
This repository contains the SPARQL queries used to validate and query the APA4CAD ontology.

##Related ontology 
Available at : https://github.com/Morgane-Ontology/APA4CAD.git

## Content
- `SPARQL.pdf`: list of SPARQL queries used in the manuscript.

- ## Purpose
These queries were designed to verify the consistency of the ontology and to illustrate its ability 
to support structured querying and reasoning.
